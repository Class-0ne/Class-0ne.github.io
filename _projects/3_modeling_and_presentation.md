## Overview

This page is meant for work where the final outcome was not only a model or study, but also a technical presentation.

## Technical Workflow

Use this section to describe:

- the model or simulation setup
- the validation process
- the interpretation of the results

## Presentation Outputs

Instead of creating a separate top-level menu for presentations, keep them attached to the project page:

- conference poster
- oral presentation slides
- summary figures
- related report

## Why this structure is useful

This keeps the site simpler.
Readers can understand the project first, then open the poster or slides as outputs of that work.

## Notes for the final version

If one presentation is especially important, make sure the project page still centers the technical work rather than the event itself.
