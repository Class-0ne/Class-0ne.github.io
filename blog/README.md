# 블로그 사용 방법

## 새 포스트 작성하기

1. `blog/posts/` 폴더에 `YYYY-MM-DD-제목.md` 형식으로 파일 생성
2. 다음 형식으로 작성:

```markdown
# 포스트 제목

**작성일:** 2025-10-12
**카테고리:** 카테고리명

여기에 본문 내용을 작성합니다.

## 부제목

- 리스트 항목
- 항목 2

### 코드 예시
\`\`\`javascript
console.log("Hello");
\`\`\`
```

3. `blog/blog.js` 파일을 열어서 `posts` 배열에 새 포스트 정보 추가:

```javascript
const posts = [
    {
        title: "새 포스트 제목",
        date: "2025-10-12",
        category: "카테고리",
        excerpt: "포스트 요약 (2-3줄)",
        file: "2025-10-12-new-post.md"
    },
    // ... 기존 포스트들
];
```

4. GitHub에 push하면 자동으로 배포됩니다!

## 파일 구조

```
blog/
├── index.html      # 블로그 목록 페이지
├── post.html       # 개별 포스트 표시 페이지
├── blog.js         # 포스트 목록 관리
├── posts/          # 마크다운 포스트 파일들
│   ├── 2025-10-12-first-post.md
│   └── 2025-10-11-welcome.md
└── README.md       # 이 파일
```

## 마크다운 문법

- `# 제목` - 큰 제목
- `## 부제목` - 중간 제목
- `**굵게**` - 굵은 글씨
- `*기울임*` - 기울임
- `[링크](URL)` - 링크
- `` `코드` `` - 인라인 코드
- 코드 블록은 \`\`\` 3개로 감싸기
